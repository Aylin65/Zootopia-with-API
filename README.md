# Zootopia with API

A Python project that generates an HTML webpage with animal information, fetched live from the [Animals API](https://api-ninjas.com/api/animals) by API Ninjas.

## Features

- User enters an animal name
- Program fetches matching animal data from the API
- Generates an HTML page with animal cards (name, diet, location, type)
- Shows an error message if the animal doesn't exist
